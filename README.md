# queue-using-array
A C++ implementation of Queue using an array. This program demonstrates basic queue operations like enqueue, dequeue, front, and isEmpty following FIFO principle.
# 📚 Queue using Array (C++)

This project implements a **Queue** data structure using an **Array** in C++.

---

## 📌 Concept

A Queue follows the **FIFO (First In First Out)** principle:

* The first element added is the first one to be removed.

---

## 🚀 Operations Implemented

* ✅ Enqueue (Insert element)
* ✅ Dequeue (Remove element)
* ✅ Front (View first element)
* ✅ isEmpty (Check if queue is empty)

---

## 🧠 Approach

* Use an array to store elements
* Maintain two pointers:

  * `front` → points to first element
  * `rear` → points to last element

---

## ⏱️ Time Complexity

| Operation | Complexity |
| --------- | ---------- |
| Enqueue   | O(1)       |
| Dequeue   | O(1)       |
| Front     | O(1)       |
